AU-tomation Toolkit 'Utilities' library

The library add some useful function blocks that ca be used for special features.


## StringBuilder
The StringBuilder function block is used to build a string from multiple parts.
It implements a fluent interface, so you can chain multiple calls to methods to build your string.
